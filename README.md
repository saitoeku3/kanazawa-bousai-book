## これは何？

大学の課題で作成したスマートフォン用の Web アプリケーションです。

## 開発手順

```bash
$ git clone git@github.com:saitoeku3/kanazawa-bousai-book.git
$ cd kanazawa-bousai-book
$ npm ci
$ npm run serve
```
